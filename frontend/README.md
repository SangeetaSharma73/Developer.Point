# Frontend Docs

1️⃣ Create a React + TypeScript Project

The easiest and modern way is using Vite (faster than Create React App).

Run:

```
npm create vite@latest frontend -- --template react
```

Select:

```txt
Framework: React
Variant: TypeScript
```

2️⃣ Run the Project

Start development server:

```bash
npm run dev
```
